To run code, please perform the following steps:

1)  Create a directory called Vinegar_MC3P2.
2)  Within the directory, download learner.py, KNNLearner.py, and LinRegLearner.py.
3)  Create a directory within Vinegar_MC3p2 called Orders.  Download orders.csv and
orderstest.csv into this directory.
4)  The project comes ready to run on ML4T-220.  Just run learner.py.
5)  To run for IBM, do a find and replace (ctrl+r) for ML4T-220 and replace it with IBM.  Then
run learner.py.
